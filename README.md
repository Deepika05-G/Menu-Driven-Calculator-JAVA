# Menu Driven Calculator (Java)

## Project Overview
This is a menu-driven calculator program developed in Java. It allows users to perform basic arithmetic operations using a console-based menu.

## Features
- Menu-based interaction
- Addition, Subtraction, Multiplication, Modulus, Division
- Loops until user chooses Exit
- Handles division by zero

## Technologies Used
- Java

## Purpose
This mini project was developed to strengthen my logical thinking and understanding of Java programming fundamentals.
